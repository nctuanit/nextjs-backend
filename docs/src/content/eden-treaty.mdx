# Eden Treaty — Type-Safe API Client

Generate a fully type-safe client for your API using Eden Treaty and the codegen CLI.

## How It Works

1. You write controllers with decorators (`@Get`, `@Post`, `@Body`...)
2. The codegen reads your source code via TypeScript Compiler API
3. It generates an Elysia type chain with **real return types** and **body schemas**
4. Eden Treaty uses these types for full autocomplete

## Generate Types

```bash
bun run eden:generate src/app.module.ts --output src/eden.ts
```

### Watch Mode

Auto-regenerate when source files change:

```bash
bun run eden:generate src/app.module.ts --output src/eden.ts --watch
```

## Generated Output

From a controller like this:

```typescript
@Controller('/todos')
export class TodosController {
  @Get('/') getAll() { return this.service.findAll(); }
  @Post('/') create(@Body() body: { title: string }) { ... }
}
```

The codegen produces:

```typescript
interface TodosItem {
  id: number;
  title: string;
  done: boolean;
}

const _app = new Elysia()
  .get("/todos", () => ({} as TodosItem[]))
  .post("/todos", () => ({} as TodosItem), {
    body: t.Object({ title: t.String() })
  });

export type App = typeof _app;
```

## Use the Client

```typescript
import { treaty } from '@elysiajs/eden';
import type { App } from './eden';

const api = treaty<App>('http://localhost:3000');

// ✅ Full autocomplete
const { data: todos } = await api.todos.get();
todos[0].title; // string ← autocomplete works!

// ✅ Body type enforced
const { data: newTodo } = await api.todos.post({
  title: 'New Todo',  // ← required by TypeScript
});
```

## Features

| Feature | Description |
|---------|-------------|
| **Return types** | Automatically inferred through service calls |
| **Body schemas** | Converted to TypeBox (`t.String()`, `t.Number()`...) |
| **Path params** | `:id` → `params: t.Object({ id: t.String() })` |
| **Optional fields** | `role?: string` → `t.Optional(t.String())` |
| **Named interfaces** | Deduplicated, reusable type definitions |
| **Watch mode** | `--watch` for auto-regeneration |

## Install Eden

```bash
bun add @elysiajs/eden
```
