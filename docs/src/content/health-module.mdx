# Health Check Module

Auto-expose a `GET /health` endpoint for monitoring and load balancers.

## Setup

```typescript
import { Module, HealthModule } from 'next-js-backend';

@Module({
  imports: [HealthModule],
})
export class AppModule {}
```

That's it! `GET /health` is now available.

## Response Format

```json
{
  "status": "ok",
  "timestamp": "2026-03-15T07:00:00.000Z",
  "uptime": 12345,
  "memory": {
    "rss": 45,
    "heapUsed": 22,
    "heapTotal": 48,
    "unit": "MB"
  }
}
```

## Custom Health Checks

Inject `HealthService` to extend health checks:

```typescript
import { Injectable, HealthService } from 'next-js-backend';

@Injectable()
export class AppService {
  constructor(private readonly health: HealthService) {}

  getStatus() {
    return this.health.check();
  }
}
```
