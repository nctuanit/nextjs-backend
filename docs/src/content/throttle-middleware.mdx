# Rate Limiting & Throttle

Control request rates per route or controller using `@Throttle()`.

## @Throttle — Per-Route Throttling

```typescript
import { Controller, Get, Throttle } from 'next-js-backend';

@Controller('/api')
export class ApiController {
  @Throttle({ limit: 5, ttl: 60 })  // 5 requests per 60 seconds
  @Get('/expensive')
  compute() {
    return { result: 'expensive computation' };
  }

  @Throttle({ limit: 100, ttl: 60 })
  @Get('/cheap')
  cheapOp() {
    return { result: 'fast' };
  }
}
```

## Class-Level Throttle

Apply to all routes in a controller:

```typescript
@Throttle({ limit: 30, ttl: 60 })
@Controller('/api')
export class ApiController {
  @Get('/a') a() { ... }  // 30 req/min
  @Get('/b') b() { ... }  // 30 req/min
}
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `limit` | `number` | `10` | Max requests in the time window |
| `ttl` | `number` | `60` | Time window in seconds |

## @UseMiddleware — Per-Route Middleware

Apply middleware to specific routes:

```typescript
import { Controller, Get, UseMiddleware, Injectable } from 'next-js-backend';

@Injectable()
class LoggingMiddleware {
  use(req: Request, res: any, next: () => void) {
    console.log(`[${new Date().toISOString()}] ${req.method} ${req.url}`);
    next();
  }
}

@Controller('/data')
export class DataController {
  @UseMiddleware(LoggingMiddleware)
  @Get('/')
  getData() {
    return { data: 'logged!' };
  }
}
```

### Class-Level Middleware

```typescript
@UseMiddleware(LoggingMiddleware, AuthMiddleware)
@Controller('/admin')
export class AdminController {
  @Get('/') dashboard() { ... }
  @Get('/users') users() { ... }
}
```

Middleware executes **before** guards, in the order specified.
