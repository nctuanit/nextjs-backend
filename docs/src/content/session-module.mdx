# Session Module

`SessionModule` cung cấp cơ chế quản lý phiên bản thân thiện và mạnh mẽ cho ứng dụng, thiết kế đặc biệt xoay quanh Cookies (mặc định là HttpOnly, giúp ngăn XSS). Nó được tích hợp chặt chẽ sẵn vào framework và lấy cảm hứng từ các tính năng Server Actions của Next.js.

## Thiết lập

Để kích hoạt Session, bạn cần đăng ký `SessionModule` trong gốc `AppModule` thông qua phương thức `register()`.

```typescript
import { Module } from './src/decorators/module.decorator';
import { SessionModule } from './src/session/session.module';
import { AppController } from './app.controller';

@Module({
  imports: [
    SessionModule.register({
      secret: 'super-secret-key-12345', // Chuỗi để mã hóa signed cookie 
      cookieName: 'my_app_sid', // (Tùy chọn) Mặc định là 'sid'
      ttl: 86400 * 7, // Hạn sử dụng: 7 ngày (Tính bằng giây)
    })
  ],
  controllers: [AppController]
})
export class AppModule {}
```

## Tương tác với SessionService

Bên trong các controller hoặc dịch vụ (provider) khác, bạn có thể inject trực tiếp `SessionService` để tạo, đọc hoặc huỷ session. Payload của session tự động quản lý cookie trả về trình duyệt.

```typescript
import { Controller, Post, Body, Injectable } from '../src';
import { SessionService } from '../src/session/session.service';

@Controller('/auth')
export class AuthController {
  constructor(private readonly sessionService: SessionService) {}

  @Post('/login')
  async login(@Body() body: any) {
    // ... kiểm tra tài khoản, mật khẩu ...
    const user = { id: 1, role: 'admin' };
    
    // Tạo session -> Elysia/NextJS Backend sẽ tự động gắn cookie phản hồi!
    const sid = await this.sessionService.createSession(user);
    
    return { message: 'Đăng nhập thành công' };
  }

  @Post('/logout')
  async logout(@Session() session: any) {
    // Hủy phiên hiện tại dựa trên SessionData trích xuất được
    if (session?.sid) { // hoặc lấy thẳng ID từ request cookie nếu bạn tự bóc tách
       // Lưu ý: @Session payload không chứa sid mà chứa cấu hình dữ liệu trả về, bạn có thể cần bóc Request.
    }
    return { message: 'Đăng xuất' };
  }
}
```

## Global Type Augmentation (Khởi tạo Cấu trúc Type An Toàn)

Theo mặc định, `@Session()` trả về `Record<string, unknown> | null`. Bạn có thể thay thế nó ở toàn cục dự án để mang lại trải nghiệm IntelliSense mượt mà bằng công cụ TypeScript Declaration Merging. 

Tạo một tệp `types.d.ts` hoặc `globals.d.ts` trong dự án Next.js của bạn:

```typescript
// types.d.ts
import 'next-js-backend';

declare module 'next-js-backend' {
  export interface SessionData {
    userId: string;
    role: 'admin' | 'user';
    preferences: { theme: string };
  }

  export interface User {
    id: number;
    email: string;
    isActive: boolean;
  }
}
```

Từ đó, trình duyệt code của bạn sẽ hiểu mọi cú pháp:
```typescript
@Get('/me')
getProfile(@Session() session: SessionData) {
   return session.role; // Tự động gợi ý 'admin' | 'user'
}
```

## Custom SessionStore (Tích hợp Redis/DB)

Mặc định, module sử dụng `InMemorySessionStore`. Do NodeJS hay Next.js Restart liên tục (đặc biệt trong Serverless), In-Memory sẽ bị mất trắng. Session Module thiết kế để bạn cắm (plug-in) các bộ nhớ bên ngoài tuân theo lớp trừu tượng `SessionStore`.

Ví dụ: Bạn có thể đưa Redis vào luồng Injection.

```typescript
// redis.store.ts
import { SessionStore, Injectable } from 'next-js-backend';
import { createClient } from 'redis';

// Phải kế thừa abstract SessionStore!
@Injectable()
export class RedisSessionStore extends SessionStore {
  private client = createClient();

  constructor() {
    super();
    this.client.connect();
  }

  async get(id: string) {
    const data = await this.client.get(`session:${id}`);
    return data ? JSON.parse(data) : null;
  }

  async set(id: string, data: any, ttl: number) {
    await this.client.setEx(`session:${id}`, ttl || 86400, JSON.stringify(data));
  }

  async destroy(id: string) {
    await this.client.del(`session:${id}`);
  }
  
  async touch(id: string, ttl: number) {
    await this.client.expire(`session:${id}`, ttl);
  }
}
```

Sau đó đăng ký vào `SessionModule`:

```typescript
SessionModule.register({
  store: RedisSessionStore, // Cung cấp Class Custom
  ttl: 3600
})
```
