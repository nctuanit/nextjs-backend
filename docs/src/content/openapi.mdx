# Tài Liệu API (OpenAPI & Swagger)

Đặc tả OpenAPI (OpenAPI Specification) là một định dạng tài liệu độc lập với ngôn ngữ, thường được sử dụng nhằm phác thảo cấu trúc giao diện cho các API RESTful.

Library hoạt động dựa trên cơ chế tích hợp mượt mà với hệ sinh thái trình cắm (plugin) đa dạng của Elysia. Thiết lập giao diện Swagger UI hay cấu trúc tài liệu OpenAPI chuyên dụng giờ đây chỉ yêu cầu cài đặt trong vài phút.

## Cài Đặt Khởi Tạo

Hãy đảm bảo rằng dự án của bạn đã cài đặt plugin `@elysiajs/swagger` trong danh sách các gói phụ thuộc (dependencies):

```bash
bun add @elysiajs/swagger
```

## Tích hợp thông qua `ElysiaFactory`

Để khai báo cấu trúc Swagger, quy trình ứng dụng tiến hành gắn (inject) plugin vào trực tiếp hệ khởi tạo của ứng dụng Elysia cơ sở ngay sau khi `ElysiaFactory` thiết lập môi trường (bootstrap). Quá trình diễn ra hoàn toàn liền mạch do factory của chúng tôi trực tiếp liên thông với tầng xử lý của động cơ Elysia. 

```typescript
import { Module, ElysiaFactory } from 'next-js-backend';
import { swagger } from '@elysiajs/swagger';
import { UsersController } from './users.controller';

@Module({
  controllers: [UsersController]
})
class AppModule {}

export const dynamic = 'force-dynamic';

export const { GET, POST, PUT, DELETE, OPTIONS } = ElysiaFactory.createNextJsHandlers(AppModule, { 
    globalPrefix: '/api' 
});

// Chú ý: Cấu hình plugin Swagger hiện tại chưa tương thích trực tiếp với luồng 
// xuất giao diện (UI) qua Route Handler của Next.js ngay lập tức. Tính năng này sắp ra mắt!
```

Khi máy chủ khởi động, một bộ giao diện Swagger UI tương tác sẽ tự động được thiết kế, trình bày hệ thống định tuyến (routes) và các lược đồ đối tượng (schemas) của toàn bộ các cấu trúc điểm cuối `@Controller`.
