# A2A Protocol (Agent-to-Agent)

**A2A (Agent-to-Agent)** là giao thức mở của Google cho phép các AI Agent giao tiếp với nhau qua HTTP. Next.js Backend tích hợp đầy đủ A2A spec, bao gồm cả JSON-RPC và SSE streaming.

> Xem thêm: [A2A Protocol Specification](https://google.github.io/A2A/)

---

## A2A Server

Expose agent của bạn như một A2A-compatible HTTP endpoint:

```typescript
import { Elysia } from 'elysia';
import { A2AServer } from 'next-js-backend/ai';

const a2aServer = new A2AServer({
  runtime: aiService.getRuntime(),
  card: {
    name: 'SupportAgent',
    description: 'Agent hỗ trợ khách hàng.',
    capabilities: { streaming: true },
    skills: [
      {
        id: 'customer-support',
        name: 'Hỗ trợ khách hàng',
        description: 'Trả lời câu hỏi về sản phẩm và dịch vụ',
      },
    ],
  },
});

// Mount vào ứng dụng Elysia
app.use(a2aServer.elysia());
```

**Các endpoint được tạo tự động:**

| Endpoint | Method | Mô tả |
|----------|--------|-------|
| `/.well-known/agent.json` | GET | AgentCard — mô tả khả năng agent |
| `/a2a` | POST | JSON-RPC 2.0 handler |
| `/a2a` (streaming) | POST | SSE stream (`tasks/sendSubscribe`) |

### JSON-RPC Methods

```typescript
// tasks/send — gửi task và chờ kết quả
POST /a2a
{
  "jsonrpc": "2.0",
  "method": "tasks/send",
  "id": "req-1",
  "params": {
    "id": "task-uuid",
    "sessionId": "user-123",
    "message": {
      "role": "user",
      "parts": [{ "type": "text", "text": "Xin chào!" }]
    }
  }
}

// tasks/get — lấy trạng thái task
{
  "method": "tasks/get",
  "params": { "id": "task-uuid" }
}

// tasks/cancel — hủy task
{
  "method": "tasks/cancel",
  "params": { "id": "task-uuid" }
}
```

## A2A Client

Gọi bất kỳ A2A-compatible agent nào từ ứng dụng của bạn:

```typescript
import { A2AClient } from 'next-js-backend/ai';

const client = new A2AClient({
  baseUrl: 'https://remote-agent.example.com',
  apiKey: process.env.REMOTE_AGENT_KEY, // optional
  timeoutMs: 30_000,
});
```

### Gửi task và chờ kết quả

```typescript
// Kiểm tra khả năng agent
const card = await client.getAgentCard();
console.log(card.name, card.capabilities);

// Chat đơn giản (gửi + chờ)
const reply = await client.chat('Xin chào, bạn có thể giúp tôi không?');
console.log(reply); // string

// Gửi task đầy đủ
const task = await client.sendTask({
  id: crypto.randomUUID(),
  sessionId: 'user-123',
  message: {
    role: 'user',
    parts: [{ type: 'text', text: 'Tìm đơn hàng của tôi' }],
  },
});
console.log(task.status.state); // 'completed'
```

### Streaming response

```typescript
for await (const event of client.chatStream('Kể một câu chuyện ngắn')) {
  if (event.type === 'text_delta') {
    process.stdout.write(event.text);
  }
}
```

### Compose nhiều agents

```typescript
@Injectable()
export class OrchestratorService {
  private readonly searchAgent = new A2AClient({ baseUrl: 'https://search-agent.example.com' });
  private readonly summaryAgent = new A2AClient({ baseUrl: 'https://summary-agent.example.com' });

  async research(topic: string): Promise<string> {
    // Agent 1: tìm kiếm
    const searchResults = await this.searchAgent.chat(`Tìm thông tin về: ${topic}`);

    // Agent 2: tóm tắt kết quả
    const summary = await this.summaryAgent.chat(`Tóm tắt:\n${searchResults}`);

    return summary;
  }
}
```

## Tích hợp với Next.js Route Handler

```typescript
// app/api/a2a/[[...a2a]]/route.ts
import { ElysiaFactory } from 'next-js-backend';
import { AppModule } from '@/modules/app.module';

export const { GET, POST } = ElysiaFactory.createNextJsHandlers(AppModule, {
  globalPrefix: '/api',
});
```

Sau đó mount `A2AServer` trong `AppModule` để agent của bạn nhận request A2A qua `/api/a2a`.

## AgentCard

AgentCard mô tả khả năng của agent để các client khác biết cách tương tác:

```typescript
{
  name: 'SupportAgent',
  description: 'Agent hỗ trợ khách hàng tiếng Việt',
  url: 'https://api.example.com/a2a',
  version: '2.0.0',
  capabilities: {
    streaming: true,
    pushNotifications: false,
    stateTransitionHistory: true,
  },
  skills: [
    {
      id: 'order-lookup',
      name: 'Tra cứu đơn hàng',
      description: 'Tìm kiếm và hiển thị trạng thái đơn hàng theo ID',
      examples: ['Đơn hàng #1234 ở đâu?', 'Tình trạng đơn hàng của tôi'],
    },
  ],
}
```
