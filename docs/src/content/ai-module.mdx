# AI Module

**AI Module** là một tích hợp AI mạnh mẽ, type-safe cho Next.js Backend. Cho phép bạn xây dựng các AI Agent thông minh, có khả năng gọi Tools, duy trì bộ nhớ hội thoại và tạo output có cấu trúc — tất cả trong kiến trúc NestJS quen thuộc.

## Tính năng

- 🤖 **AI Agents** — Vòng lặp agentic hoàn chỉnh (prompt → LLM → tool call → response)
- 🔧 **Tool Registry** — Đăng ký công cụ bằng decorator `@Tool`, tự động validate input với TypeBox
- 🧠 **Memory System** — Lưu lịch sử hội thoại (InMemory hoặc Redis)
- 📐 **Structured Output** — Tạo JSON type-safe với retry tự động khi validation lỗi
- 🔄 **Workflow Engine** — Thực thi quy trình nhiều bước với `@Workflow` và `@Step`
- 🧩 **Plugin System** — Mở rộng agent với SearchPlugin, DatabasePlugin
- 🌐 **A2A Protocol** — Giao tiếp agent-to-agent theo chuẩn Google A2A

## Providers hỗ trợ

| Provider | Model ví dụ |
|----------|------------|
| `openai` | `gpt-4o`, `gpt-4-turbo`, `o1` |
| `anthropic` | `claude-3-5-sonnet-20241022`, `claude-3-haiku` |
| `google` | `gemini-2.0-flash`, `gemini-1.5-pro` |

> Tất cả providers dùng native `fetch` API — **zero external SDK dependency**.

## Cài đặt

```typescript
import { Module, AiModule } from 'next-js-backend';

@Module({
  imports: [
    AiModule.register({
      providers: [
        { provider: 'openai', apiKey: process.env.OPENAI_API_KEY! },
      ],
      tools: [UserTools, ProductTools],
      agents: [SupportAgent],
    }),
  ],
})
export class AppModule {}
```

## Định nghĩa Tool

```typescript
import { Injectable, Tool } from 'next-js-backend';
import { Type as t } from '@sinclair/typebox';

@Injectable()
export class UserTools {
  @Tool({
    name: 'get_user',
    description: 'Lấy thông tin người dùng theo ID',
    schema: t.Object({ id: t.String() }),
  })
  async getUser(id: string) {
    // return from database
    return { id, name: 'Nguyen Van A', email: 'a@example.com' };
  }

  @Tool({
    name: 'search_users',
    description: 'Tìm kiếm người dùng theo tên',
    schema: t.Object({ query: t.String(), limit: t.Optional(t.Number()) }),
  })
  async searchUsers(query: string, limit = 10) {
    return []; // query DB
  }
}
```

> **Validate input tự động**: Nếu `schema` được cung cấp, TypeBox sẽ validate và coerce input trước khi gọi handler. Nếu invalid, `ToolInputValidationError` sẽ được throw.

## Sử dụng AiService

```typescript
import { Injectable, AiService } from 'next-js-backend';

@Injectable()
export class ChatService {
  constructor(private readonly ai: AiService) {}

  // Chat đơn giản
  async chat(message: string) {
    return this.ai.run('SupportAgent', message);
  }

  // Chat có nhớ lịch sử (theo sessionId)
  async chatWithMemory(message: string, sessionId: string) {
    return this.ai.run('SupportAgent', message, { sessionId });
  }

  // Streaming
  async *streamChat(message: string) {
    for await (const chunk of this.ai.stream('SupportAgent', message)) {
      if (chunk.type === 'text') yield chunk.content;
    }
  }
}
```

## Structured Output (Typed Response)

Dùng `runTyped` để nhận response đã được validate theo TypeBox schema:

```typescript
import { Type as t } from '@sinclair/typebox';

const ProductSchema = t.Object({
  name: t.String(),
  price: t.Number(),
  category: t.String(),
  inStock: t.Boolean(),
});

const result = await this.ai.runTyped('ProductAgent', 'Tìm sản phẩm iPhone', {
  schema: ProductSchema,
  temperature: 0,     // deterministic
  retries: 2,         // tự retry nếu JSON lỗi
});
// result: { name: string; price: number; category: string; inStock: boolean }
```

## Options đầy đủ của AiModule.register()

```typescript
AiModule.register({
  // Danh sách LLM providers
  providers: [
    { provider: 'openai', apiKey: '...' },
    { provider: 'anthropic', apiKey: '...' },
    { provider: 'google', apiKey: '...' },
    // Custom baseUrl (ví dụ: Azure OpenAI)
    { provider: 'openai', apiKey: '...', baseUrl: 'https://my-azure.openai.azure.com' },
  ],

  // Tool classes (được inject vào tất cả agents)
  tools: [UserTools, ProductTools, OrderTools],

  // Agent classes
  agents: [SupportAgent, SalesAgent],

  // Memory store (mặc định: InMemoryMemoryStore)
  memory: new RedisMemoryStore({ client: redisClient, ttl: 3600 }),

  // Plugins mở rộng khả năng agent
  plugins: [
    new SearchPlugin({ apiKey: process.env.TAVILY_API_KEY! }),
    new DatabasePlugin({ query: (sql) => db.$queryRawUnsafe(sql), allowedTables: ['users'] }),
  ],
})
```
