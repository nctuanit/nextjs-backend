# NextAuth (Auth.js v5)

Integrate Auth.js v5 for OAuth, credentials, and session-based authentication.

## Setup

```bash
bun add @auth/core
```

```typescript
import { Module, NextAuthModule } from 'next-js-backend';
import GitHub from '@auth/core/providers/github';

@Module({
  imports: [
    NextAuthModule.forRoot({
      providers: [
        GitHub({
          clientId: process.env.GITHUB_CLIENT_ID!,
          clientSecret: process.env.GITHUB_CLIENT_SECRET!,
        }),
      ],
      secret: process.env.AUTH_SECRET!,
      trustHost: true,
    }),
  ],
})
export class AppModule {}
```

## Protect Routes

Use `NextAuthGuard` to require authentication:

```typescript
import { Controller, Get, UseGuards } from 'next-js-backend';
import { NextAuthGuard } from 'next-js-backend';

@Controller('/dashboard')
export class DashboardController {
  @UseGuards(NextAuthGuard)
  @Get('/')
  getProfile() {
    return { message: 'Authenticated!' };
  }
}
```

## Access Session

The guard injects the session user into the request context:

```typescript
@UseGuards(NextAuthGuard)
@Get('/me')
getMe(@Req() req: any) {
  return req.user; // Session user data
}
```

## Auth Endpoints

NextAuth automatically registers these routes:

| Route | Description |
|-------|-------------|
| `GET /api/auth/signin` | Sign-in page |
| `GET /api/auth/signout` | Sign-out |
| `GET /api/auth/session` | Get session |
| `GET /api/auth/callback/:provider` | OAuth callback |

## Environment Variables

```bash
GITHUB_CLIENT_ID=your_client_id
GITHUB_CLIENT_SECRET=your_client_secret
AUTH_SECRET=your_random_secret
```
