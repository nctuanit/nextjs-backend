# AI Plugins & Testing

## Plugin System

Plugin cho phép mở rộng khả năng của AI Agent bằng các tool được inject tự động vào **tất cả** agents. Khác với Tool thông thường (gắn với một agent cụ thể), plugin tools luôn khả dụng toàn cục.

### SearchPlugin (Tavily)

Tích hợp tìm kiếm web bằng [Tavily API](https://tavily.com):

```typescript
import { SearchPlugin } from 'next-js-backend/ai';

AiModule.register({
  plugins: [
    new SearchPlugin({ apiKey: process.env.TAVILY_API_KEY! }),
  ],
  // ...
})
```

**Tools được đăng ký:**
- `web_search(query: string)` — Tìm kiếm web, trả về danh sách kết quả với title, url, content
- `get_webpage(url: string)` — Lấy nội dung của một trang web

### DatabasePlugin

Cho phép agent query database an toàn (read-only):

```typescript
import { DatabasePlugin } from 'next-js-backend/ai';

AiModule.register({
  plugins: [
    new DatabasePlugin({
      // Hàm execute SQL — truyền vào DB client của bạn
      query: (sql) => db.$queryRawUnsafe(sql),

      // Whitelist bảng được phép truy cập
      allowedTables: ['users', 'products', 'orders'],

      // Mô tả schema cho agent (tùy chọn)
      schemaDescription: 'users(id, name, email), products(id, name, price)',
    }),
  ],
})
```

**Tools được đăng ký:**
- `list_tables()` — Liệt kê các bảng được phép
- `query_database(sql: string)` — Thực thi SELECT query

> **Bảo mật**: Plugin tự động chặn các câu lệnh nguy hiểm: `UNION`, `WITH`, block comment `/**/`, `INFORMATION_SCHEMA`, `xp_`, `sys.`, `pg_`, stacked queries (`;`), v.v.

### Custom Plugin

Implement interface `AIPlugin` để tạo plugin tùy chỉnh:

```typescript
import type { AIPlugin, ToolRegistry } from 'next-js-backend/ai';

export class WeatherPlugin implements AIPlugin {
  readonly name = 'weather';

  async register(registry: ToolRegistry): Promise<void> {
    registry.registerTool({
      name: 'get_weather',
      description: 'Lấy thông tin thời tiết cho một thành phố',
      parameters: {
        type: 'object',
        properties: {
          city: { type: 'string', description: 'Tên thành phố' },
        },
        required: ['city'],
      },
      instance: this,
      handler: async (city: unknown) => {
        const res = await fetch(`https://wttr.in/${city}?format=j1`);
        return res.json();
      },
    });
  }
}

// Đăng ký:
AiModule.register({
  plugins: [new WeatherPlugin()],
})
```

## AI Testing Utilities

Test AI agents mà không cần gọi API thực tế bằng `AiTestBed` và `MockLLMProvider`.

### AiTestBed

```typescript
import { AiTestBed } from 'next-js-backend/ai/testing';

describe('SupportAgent', () => {
  it('tìm kiếm user và trả về kết quả', async () => {
    const bed = AiTestBed.create({
      agents: [SupportAgent],
      tools: [new UserTools()],
      mockProvider: {
        responses: [
          // Lần 1: agent quyết định gọi tool
          { toolCall: { name: 'get_user', args: { id: '1' } } },
          // Lần 2: agent trả lời sau khi có kết quả tool
          'Đây là thông tin người dùng ID 1: Nguyen Van A.',
        ],
      },
    });

    const result = await bed.run('SupportAgent', 'Tìm user có id là 1');
    expect(result).toContain('Nguyen Van A');
    expect(bed.provider.callCount).toBe(2);
  });

  it('stream response', async () => {
    const bed = AiTestBed.create({
      agents: [SupportAgent],
      mockProvider: { responses: ['Xin chào! Tôi có thể giúp gì cho bạn?'] },
    });

    let output = '';
    for await (const chunk of bed.stream('SupportAgent', 'Xin chào')) {
      if (chunk.type === 'text') output += chunk.content;
    }
    expect(output).toBeTruthy();
  });
});
```

### MockLLMProvider

```typescript
import { MockLLMProvider } from 'next-js-backend/ai/testing';

const provider = new MockLLMProvider({
  responses: [
    // Text response
    'Xin chào!',

    // Tool call response
    { toolCall: { name: 'search_users', args: { query: 'admin' } } },

    // Lỗi có kiểm soát
    new Error('Rate limit exceeded'),
  ],
});

// Assertions sau khi chạy
provider.callCount;         // số lần LLM được gọi
provider.lastMessages;      // messages của lần gọi cuối
provider.reset();           // reset về trạng thái ban đầu
```

### Kiểm tra xem tool có được gọi đúng không

```typescript
// Mock tool để verify
class MockUserTools {
  getUser = jest.fn().mockResolvedValue({ id: '1', name: 'Test User' });
}

// Dùng trong AiTestBed
const mockTools = new MockUserTools();
const bed = AiTestBed.create({
  agents: [SupportAgent],
  tools: [mockTools],
  mockProvider: {
    responses: [
      { toolCall: { name: 'get_user', args: { id: '1' } } },
      'Found user: Test User',
    ],
  },
});

await bed.run('SupportAgent', 'Find user 1');
expect(mockTools.getUser).toHaveBeenCalledWith('1');
```
