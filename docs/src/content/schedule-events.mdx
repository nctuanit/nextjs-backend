# Task Scheduling & Events

## @Cron — Task Scheduling

Schedule methods to run on a cron schedule using the `@Cron()` decorator. Powered by [croner](https://github.com/hexagon/croner).

### Setup

Import `ScheduleModule` in your app module:

```typescript
import { Module, ScheduleModule } from 'next-js-backend';

@Module({
  imports: [ScheduleModule],
  providers: [TasksService],
})
export class AppModule {}
```

### Usage

```typescript
import { Injectable, Cron } from 'next-js-backend';

@Injectable()
export class TasksService {
  @Cron('0 0 * * *')
  async dailyCleanup() {
    console.log('Running daily cleanup...');
  }

  @Cron('0 * * * *', 'hourly-sync')
  async hourlySync() {
    console.log('Syncing data...');
  }
}
```

### Cron Expressions

| Expression | Schedule |
|-----------|----------|
| `0 * * * *` | Every hour |
| `0 0 * * *` | Every day at midnight |
| `0 9 * * 1-5` | Weekdays at 9 AM |
| `0 0 1 * *` | First day of every month |

## @OnEvent — Event Emitter

Internal pub/sub event system for decoupled communication between services.

### Setup

```typescript
import { Module, EventEmitterModule } from 'next-js-backend';

@Module({
  imports: [EventEmitterModule],
  providers: [UserService, NotificationService],
})
export class AppModule {}
```

### Emit Events

```typescript
import { Injectable, EventEmitterService } from 'next-js-backend';

@Injectable()
export class UserService {
  constructor(private readonly eventEmitter: EventEmitterService) {}

  async createUser(data: { email: string }) {
    // ... create user logic
    await this.eventEmitter.emit('user.created', data);
  }
}
```

### Listen to Events

```typescript
import { Injectable, OnEvent } from 'next-js-backend';

@Injectable()
export class NotificationService {
  @OnEvent('user.created')
  async sendWelcomeEmail(data: { email: string }) {
    console.log(`Sending welcome email to ${data.email}`);
  }
}
```

### Multiple Handlers

Multiple services can listen to the same event:

```typescript
@OnEvent('order.placed')
async sendReceipt(order: any) { ... }

@OnEvent('order.placed')
async updateInventory(order: any) { ... }
```
